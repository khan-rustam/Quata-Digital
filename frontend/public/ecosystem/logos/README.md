# Product logos

The site references each product's logo via the `logo` field in
`frontend/lib/ecosystem.ts`. To swap a logo, drop the new file into this
folder and update the matching `logo:` line in `ecosystem.ts`.

Current files:

- `Quata-Pay.png`        → QUATAPAY
- `Abaqwa-logo.png`      → ABAQWA
- `Quata-Food.png`       → QUATAFOOD
- `Quata-88Bricks.png`   → 88BRICKZ
- `Quata-Mall.png`       → O3MALL
- `88basket.svg`         → 88BASKET (placeholder — replace with real artwork)
- `qmediq.svg`           → QMEDIQ   (placeholder — replace with real artwork)

Recommended export:

- Square canvas, 512×512 minimum, transparent background.
- Single-colour or gradient mark works best on dark hero panels.
- Keep ≥ 12% padding inside the canvas so the mark doesn't touch edges.
